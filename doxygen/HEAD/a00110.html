Detailed Description

template<typename U>
class roboptim::Map< U >

Apply a function several times to an input vector.

Input: [x_0^0 x_1^0 ... x_N^0 ... x_0^M x_1^M ... x_N^M]

Output: [f(x_0^0 x_1^0 ... x_N^0) ... f(x_0^M x_1^M ... x_N^M)]
